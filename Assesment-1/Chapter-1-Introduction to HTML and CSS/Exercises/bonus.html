HTML & CSS Quick Reference Guide

HTML Tags

<h1> to <h6>

Defines HTML headings. <h1> is the highest level, and <h6> is the lowest.

<h1>This is a Heading</h1>

<p>

Defines a paragraph.

<p>This is a paragraph.</p>

<a>

Defines a hyperlink.

<a href="https://www.example.com">This is a link</a>

<img>

Defines an image.

<img src="image.jpg" alt="Description">

<ul> and <li>

Defines an unordered list and list items.

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<ol> and <li>

Defines an ordered list and list items.

<ol>
    <li>First item</li>
    <li>Second item</li>
</ol>

<div>

Defines a division or section in an HTML document.

<div>This is a division.</div>

<span>

Defines a section in a document, used for styling.

<span style="color: red;">This is a span.</span>

<form>

Defines an HTML form for user input.

<form>
    <input type="text" placeholder="Enter text">
    <input type="submit" value="Submit">
</form>

<table>

Defines a table.

<table>
    <tr>
        <th>Header</th>
        <th>Header</th>
    </tr>
    <tr>
        <td>Data</td>
        <td>Data</td>
    </tr>
</table>

CSS Properties

color

Sets the color of text.

p {color: blue;}

background-color

Sets the background color of an element.

div {background-color: lightgray;}

font-size

Sets the size of the font.

h1 {font-size: 24px;}

margin

Sets the margin area on all four sides of an element.

div {margin: 20px;}

padding

Sets the padding area on all four sides of an element.

div {padding: 15px;}

border

Sets the border around an element.

div {border: 1px solid black;}

text-align

Sets the horizontal alignment of text.

h2 {text-align: center;}

display

Specifies the display behavior of an element.

div {display: flex;}

font-weight

Sets the weight (boldness) of the font.

p {font-weight: bold;}

line-height

Sets the height of a line box.

p {line-height: 1.5;}
